About This Project:

This Online FIR System was developed in 2010-2011 as a student final year project at the Institute of Information & Communication Technology (IICT), University of Sindh, Jamshoro, Pakistan. The project simulates a digital platform for filing First Information Reports (FIRs) using PHP and MySQL.

The system was designed and developed by Khalid Hussain, who also served as technical mentor and web developer, providing backend and interface support to guide the students in successfully delivering the project.

This project is now published on GitHub for archival and educational reference only. It includes legacy components such as Flash and older PHP practices, and is not intended for live or production deployment.
